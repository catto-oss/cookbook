# `cookbook`

</div>

`cookbook` is a collection of re-usable GitHub workflows for your [actions](https://docs.github.com/en/actions)
To add your own workflows, open a [PR](https://github.com/catto-oss/cookbook/pulls)!

## Workflows
* [crystal-release.yml](https://github.com/catto-oss/cookbook/blob/main/workflows/crystal-release.yml): A configurable workflow for building and releasing Crystal-based applications using `crystal build` across MacOS and Linux targets.
* [rust-release.yml](https://github.com/catto-oss/cookbook/blob/main/workflows/rust-release.yml): A configurable workflow for building and releasing Rust-based applications across MacOS, Linux, and Windows targets.
